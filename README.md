# Road-Accident-Analysis-Power_BI

This project analyzes road accident data using Power BI. 
The dashboard provides insights into accident severity, road types, weather conditions, speed limits, and light conditions.

Key Insights:
- Total Accidents: 2M
- Total Casualties: 3M
- Serious Accidents: 286K
- Fatal Accidents: 26K

Tools Used:
- Power BI
- Excel Dataset
- Data Visualization

This dashboard helps identify accident patterns and improve road safety analysis.
